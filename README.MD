# EXPRESS API WITH DOCKER COMPOSE

## HOW TO RUN API?

### IN Project DIR

#### STEP ONE

- execute: docker compose --build

### STEP TWO

- update .env for connect db in docker container

#### STEP THREE

- TEST THE API IN localhost:3000

#### Utilized

- TypeScript
- Express
- TypeORM ( no Migrations )
- PostgreSQL
- camelCase
- Docker - Docker Compose
